# Интересные материалы по математике

За время, пока я занимался кружком для учеников, а также за время обучения в университете у меня накопилось достаточно интересных практических приложений математики, которыми бы я хотел поделиться со школьниками и учителями математики. Первым это поможет глубже окунуться в удивительный мир и почувствовать близость к реальной жизни всех проходимых на уроках темах, а вторым -- откроет, возможно, новую сторону преподавания математики -- практично.

Ведь один из первых вопросов при изучении любой темы по математике -- зачем? Цикл листков по какой-то конкретной теме ставит целью попытаться ответить на этот вопрос.

Каждая маленькая тема -- отдельный `.pdf` файл.
